#  MERN Todo App with Authentication

A full-stack **Todo Application** built using the **MERN stack** (MongoDB, Express, React, Node.js) with **user authentication** and **CRUD operations**.


- User Registration & Login (JWT Authentication)
- Create, Read, Update, Delete Todos
- Protected Routes (Auth required)
- MongoDB Database
- RESTful APIs
- Responsive React UI

---

# Tech Stack

 Frontend
- React JS
- Axios
- React Router DOM
- CSS / Material UI (optional)

# Backend
- Node.js
- Express.js
- MongoDB & Mongoose
- JWT (JSON Web Token)
- bcryptjs
- dotenv
- cors
